Chapter 01, Section 02, Exercise 550_FontProportions.cpp


Draw a box around the letters.


In this exercise, you will learn:


● Where the bounds (upper left corner through to the bottom right corner) of characters drawn in the drawing area are located.


To do this exercise you need to know:


● How to load the fwc project in Qt Creator, and run it.

● What functions and parameters are.  ( See Exercise 010_ChangeText.cpp )


Notice that this is a 500-level exercise!


500-level exercises are exercises that belong in the section they are in, but might require more experience and practice in other sections before you are ready to try them.  This particular exercise is demonstrating that there is some extra space in how characters are drawn in the drawing area that you will have to account for in later exercises when you are drawing text, in order to make the text line up properly on the screen.


You can try this exercise now, or you can wait until later exercises when you will be prompted to return to it, in order to make this issue of how drawn text has some extra spacing around it.


When you are done working on the exercise, or if you get stuck and can't figure it out, check the solution code, which lives one directory below your exercise code in the "Solutions" directory, and is named:

● 050_ChangeColorSOLN.cpp

Always understand the SOLN code before you proceed to the next exercise!


When you are ready, you can proceed to Section03: Types and Variables.
