Chapter 01, Section 02, Exercise 400_OnYourOwn.cpp


Your first On Your Own (OYO) exercise!


In this exercise, you will learn:


● How to write a simple program from scratch on your own.


To do this exercise you need to know:


● How to load the fwc project in Qt Creator, and run it.

● What functions and parameters are.  (See Exercise 010_ChangeText.cpp)

● How to use fwcDrawCircle() for the head.  (See Exercise 080_DrawCircles.cpp)

● How to use fwcDrawRectangle() for the arms, legs, and torso. (See Exercise 090_DrawRectangles.cpp)

● How to use fwcDrawText() to display the text in the drawing area.  (See Exercise 200_DrawingText.cpp)

● How to use seeout to display the text in the seeout text output area.  (See Exercise 300_SeeoutIntro.cpp)


Starting from scratch, add code to your exercise to draw the happy person and display both lines of text.  You may add your code before or after the comments, it doesn't really matter since comments are ignored by the compiler.


When you are done working on the exercise, or if you get stuck and can't figure it out, check the solution code, which lives one directory below your exercise code in the "Solutions" directory, and is named:

● 400_OnYourOwnSOLN.cpp

Always understand the SOLN code before you proceed to the next exercise!


When you are ready, you can proceed to 520_ColorNames.cpp.
