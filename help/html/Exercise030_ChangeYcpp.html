Chapter 01, Section 02, Exercise 030_ChangeY.cpp


In this exercise, you will learn:


● How to change the y coordinate that controls the vertical position for drawing on the runtime exercise draw area.


To do this exercise you need to know:


● How to load the fwc project in Qt Creator, and run it.

● What functions and parameters are.  ( See Exercise 010_ChangeText.cpp )


This exercise is similar to the last one, except that instead of changing the horizontal postion of your text, you are changing the vertical position.  Change the y value to make the exercise draw in the same location in your drawing area as where the solution draws on it's side of the window.  In this case, the y value is 50, the third parameter.  Remember to close the fwc window before you try to run your program again in Qt Creator.


Whoa!  I thought Y gets bigger as it goes up!  What gives?


Back in algebra or geometry class, y does get bigger as it goes upwards.  But the way the Qt library does graphics, y gets bigger as it goes downwards.


That's because we orient ourselves to the top left corner of our window, because windows can resize, and everything else on the window is relative to that top left corner.  Go ahead and resize a window and you will see that all of any window's content is located with respect to that top left corner.


So graphically, that top left corner is the anchor of the whole graphical circus.  If y got bigger going up from that top left corner, then we would constantly be drawing with negative numbers for y.  We could do this, but it's an extra character to type the "-" character, and we'd be typing it constantly.  Plus, it seems kind of weird to have x be positive numbers, but y be negative for how we locate things on the screen.  So the people who made Qt just flipped the y-axis upside down from how it works in algebra and geometry class.  And Qt is not the only drawing toolkit that does this, other toolkits do as well.  (But not all, some locate y=0 at the lower left corner of the drawing area, I believe.)


If you have done (or are doing) lots of algebra, geometry or other work where you are doing mathematical graphing, this make take a little getting used to, but soon it will be second nature.
