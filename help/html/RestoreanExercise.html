Restore an Exercise

FeetWetCoding Help ››

You might hose your exercise code to the point where you want to start over from the original exercise code.  We have talked about ways to make this easier, but at this time, the way to do it is to copy the original file for the exercise you are working on from the feetwetcoding .zip file that you downloaded.  If you don't have that .zip file anymore, you can just download it again by clicking the ZIP button on this page:


● https://github.com/FeetWetCoding/FeetWetCoding


...however you should be aware that the exercise might have changed since you last downloaded feetwetcoding, so be sure to run the exercise and see if it is different from what you had been working on.  Even if it has changed, it should essentially still cover the same learning material you were working on.


Also, realize that sometimes, as we are developing this software, the exercise numbers change.  So for example


● 400_C01S05_OYO5.cpp


...recently (2012Mar) changed to:


● 480_C01S05_OYO5.cpp


The exercise code is still the same, it just has a different number now.
