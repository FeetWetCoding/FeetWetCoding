Chapter 01, Section 02, Exercise 100_DrawDots.cpp


Rearrange the dots to look like the solution.


In this exercise, you will learn:


● How to draw dots on the screen using fwcDrawLine(), fwcDrawCircle() and fwcDrawRectangle().


To do this exercise you need to know:


● How to load the fwc project in Qt Creator, and run it.

● What functions and parameters are.  ( See Exercise 010_ChangeText.cpp )


Change your code from drawing the three dots in a triangular arrangement to having them all drawn on a line where y=200.


Your exercise code draws three dots in a triangle.   The green dot is on top and is harder to see.  I'm calling them "dots" here because the circle and rectangle functions are not really drawing single pixels.  The circle in this case has a radius of 1, and the rectangle has a size, so those functions are drawing more than one pixel.  The line function is drawing what appears to be a single pixel (the green dot).


For your learning, most of the time you won't be drawing single pixels like this.  Shapes like lines, circles and rectangles, and drawing images (which we will get to in exercise 250_DrawingImages.cpp) will be more useful.  But we are including this way of drawing dots and pixels on the screen so that you know how to do it and can play around with it in your own coding experiments, and there will be some exercises where we do draw pixels and dots on the screen.


Can you figure out how to make your drawing area black, so that the dots show up more clearly?  If you can't think of it, you can check the solution code.


When you are done working on the exercise, or if you get stuck and can't figure it out, check the solution code, which lives one directory below your exercise code in the "Solutions" directory, and is named:

● 100_DrawDotsSOLN.cpp

Always understand the SOLN code before you proceed to the next exercise!
