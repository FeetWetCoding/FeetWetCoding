Chapter 01, Section 02, Exercise 060:  ChangeFontSize


In this exercise, you will learn:


  • How to change the font size of text you draw in the drawing area.
  • The fontsize parameter determines the fontsize of the text you draw.
  • The fontsize parameter is optional.


To do this exercise you need to know:


  • How to load the fwc project in Qt Creator, and run it.


Once again, in this exercise, we are just changing a parameter in the function fwcDrawText().  In this case, we are changing the fifth parameter, fontsize.


Now that you have had some practice with changing parameters and running the fwc software, this documentation is going to start assuming that you have an understanding of topics that you have covered, so that the documentation doesn't repeat itself needlessly and get all bloated and difficult to read.


We are assuming now that you understand the basic idea of what a parameter is, and how to change it.  So from now on, we will just say, "Change the ____ parameter."
