The "FeetWetCoding Qt Creator Workflow" or How to use FeetWetCoding


"Workflow" here just means "how you work through the FeetWetCoding exercises in Qt Creator."  In other words, we're talking about how you load up the first FeetWetCoding exercise in Qt Creator and run it, and then load up the next exercise and run that, and so on.  It may sound a little complicated trying to describe the process here, but when you do it a few times, you will see that it is easier than this description might make it sound.


The exercises for Chapter 1 are located in Sources > C_Basics  within Qt Creator when you load the FeetWetCoding project (FeetWetCoding.pro).  The exercises for each chapter are located in their respective Chapter folder.  Each exercise also has a "solution" counterpart, which is the original exercise name with the characters "SOLN" after it.


The idea for doing exercises in FeetWetCoding is that you work on one exercise, and try to accomplish the goal of that exercise, and then if you get stuck, or if you finish the exercise, you can check your work in the "<exercise name>SOLN" file.  To work with a particular exercise you have to "tell" main.cpp which exercise you want to work on.  These instructions explain how to do that.  Don't worry if this doesn't seem to make sense right now, just learn the process, and it will start to feel natural soon.


From now on, we will refer to the location where you unzipped the FeetWetCoding download as "~FeetWetCoding".  You pronounce ~FeetWetCoding as "FeetWetCoding home."  ~FeetWetCoding will usually be Libraries\Downloads on your hard drive , which is actually:


  • C:\\Users\<your username>\Downloads


Open the HandsDirty project in Qt Creator by clicking on the FeetWetCoding.pro file in ~FeetWetCoding:





...you can also start Qt Creator up and navigate to the FeetWetCoding.pro file that way.  Once you have the FeetWetCoding project open in Qt Creator, you need to type the name of the exercise you want to work with in the appropriate place in the file main.cpp.  The next image shows what we are talking about, but don't start working on the exercises at this time.  Just make the changes we talk about here in this tutorial.


Qt Creator has a number of different views, including, "Edit", "Debug", "Projects", "Analyze" and "Help."  You will be spending most of your time while learning to write C++ code in the "Edit" view.


TIP: If you ever find yourself trying to see your code, and you do not see it on the right hand side of the screen, make sure you are in the "Edit" view by clicking on the "Edit" icon on the left side of Qt Creator. Sometimes you will be editing code, and then you will change the view, and won't be able to see your code.  Clicking on "Edit" should fix it, and your code should be viewable again.

When you click the green "Run" arrow button, Qt Creator will "build" and then run the FeetWetCoding program.  This may take a minute or more, but eventually the build should finish and the program should run.  The first time you run FeetWetCoding, when the program runs and pops up in a new window, it should look like this:





That previous window picture is the output of the IntroExercise that is currently located in file a0005.cpp.


So the IntroExercise ran successfully (I hope!) and now you are going to check the solution.  That means you are going to tell main.cpp the next exercise you want to run.  Every exercise file has a counterpart "solution" file paired with it.  The solution file for a0005.cpp is called a0005soln.cpp.  You just ran the IntroExercise which is located in a0005.cpp, and now you want to run the IntroExerciseSoln (the solution to the exercise) which is located in a0005soln.cpp.


Doubleclick on the file a0005soln.cpp so you can view the C++ code it contains.  Then highlight the exercise name, IntroExerciseSoln, and CTRL-C to copy that text:





Now open main.cpp in the Edit view (you can just double click on main.cpp in the project files list) and paste (CTRL-V) the IntroExerciseSoln text into the correct place, or you can just type it in:
