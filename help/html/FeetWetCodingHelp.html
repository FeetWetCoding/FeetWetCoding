FeetWetCoding


FeetWetCoding is a open-source software project that tries to make learning C++ fun using Qt.


● Screenshots of FeetWetCoding in action.


For more information about what this project is and why it was made, you may read About FeetWetCoding.  Here's what you need to use our software: Requirements.  This project is "alpha" software--it is currently in development and is unfinished!


Here is how to get started!


1. Get the software.

2. Learn how to use it.

3. Do the Exercises.


Weekly FeetWetCoding Google+ Video Chat Hangout


The FeetWetCoding developers host two weekly Google+ video chat hangouts.  G+ hangouts provide screen-sharing, so that we can demonstrate how to use the software and answer questions.  Anyone is welcome to join us!  You will need a Google+ account, and will need to install the hangout video chat plug-in.  From your Google+ account, add +FeetWetCoding to your circles, and you will then see when the hangout is active if you are logged in to your G+ account.


If you do not have a microphone/webcam you can still join, but you will have to text chat in the hangout from your keyboard.  We will be sure to stay in the hangout for at least 30 minutes from the start time.
