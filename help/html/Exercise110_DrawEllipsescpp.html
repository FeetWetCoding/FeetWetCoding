Chapter 01, Section 02, Exercise 110_DrawEllipses.cpp


Rearrange the ellipses to look like the solution.


In this exercise, you will learn:


● How to draw ellipses using the fwcDrawEllipse() function.


To do this exercise you need to know:


● How to load the fwc project in Qt Creator, and run it.

● What functions and parameters are.  ( See Exercise 010_ChangeText.cpp )


In this exercise you will change the x,y position, linewidth, and fill (solid) of the ellipses to match the solution.


Sample code:

   // USAGE: fwcDrawEllipse(x, y, width, height, color, linewidth, solid);
   // (solid is an optional parameter, you can use it or leave it out.)
   fwcDrawEllipse(100, 200, 10, 300, BLUE, 1);


In the USAGE comment, you will see the parameters that fwcDrawEllipse() accepts.  x,y determine the center of the ellipse, in the same way that they determine the center of the circle with fwcDrawCircle() in other exercises.  However fwcDrawCircle() only uses a radius, while fwcDrawEllipse() uses width and height to allow you to "squash" the circle so it is no longer perfectly round.  linewidth and solid work as they do with other fwc drawing tools.


What is important here is not that your exercise code can match the solution pixel-for-pixel, but that you get the right basic position, orientation, linewidth and fill, so that the blue circle is on top and is filled, the red circle is in the middle and has a fatter line, and the black circle is on the bottom with a skinny line, approximately as it is in the SOLN.


When you are done working on the exercise, or if you get stuck and can't figure it out, check the solution code, which lives one directory below your exercise code in the "Solutions" directory, and is named:

● 110_DrawEllipsesSOLN.cpp

Always understand the SOLN code before you proceed to the next exercise!


When you are ready, you can proceed to 200_DrawingText.cpp.
