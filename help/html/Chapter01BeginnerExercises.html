Beginner Exercises


This chapter contains the beginning exercises, and is where new programmers should start.


When you see something like "C01S01" it means Chapter 01, Section 01.  The exercises are named by Chapter and Section, so that they are organized in a useful, comprehensible way.


Chapter 01 Beginner Exercises introduces the basic concepts and elements of C++.  Once you have worked with the exercises in this chapter, you can move on and start working on Chapter 02 Intermediate Exercises. (Chapter 02 TOTALLY has not been written yet.  If we get to it in 2012 I will be amazed... so don't hold your breath! :-P  Also, we're talking about going from Chapter 01 directly to Chapter 05 Beginner Qt, as that makes more sense to get the user up and running on Qt directly rather than futzing about with our fwc tools... we're still thinking about how to do it. )


You NEED to read the help page How to Use FeetWetCoding to understand how to use the FeetWetCoding exercises!   And if you haven't already installed Qt Creator, you NEED to read the Install QtCreator help page and follow the instructions there or you will not be able to use FeetWetCoding at all.   If you have not read those pages and completed the instructions there,  please do so now.  Don't worry:  I'll wait here.  :-)
