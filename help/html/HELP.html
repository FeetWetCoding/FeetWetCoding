FeetWetCoding is a free, open-source software project that tries to make learning C++ fun with the Qt development platform.  For more information about what FeetWetCoding is and why it was made, you may read the Introduction.  (The intro is a long-winded mess right now--I'm going to fix it soon.)


NOTE: This documentation is still being developed--it is incomplete and in some places, inaccurate!


It is important that you understand how to use FeetWetCoding correctly.  Save yourself time and aggravation by reading the following two pages of instructions:


1. Download and install the QtCreator SDK.

2. How to use the FeetWetCoding learning framework.


Once you have read those, you are ready to get started with:


3. The Exercises.


If you are enjoying this project, you might consider downloading a new copy every few weeks to get the latest exercises:  as of early 2012, we are actively trying to add exercises and improve the documentation, and hopefully it will grow quickly.  :-)


gittown


We are also working on gittown, a git repository designed to help us (and maybe others) learn to use git in a fun way.  As of 1Feb2012, it is just a few files.  The framework of what we want to do with it is not there yet, and it is not yet helpful to anyone trying to learn git.  But soon!  :-)
