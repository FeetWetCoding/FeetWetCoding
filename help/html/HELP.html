FeetWetCoding


This project tries to make learning C++ fun.  For more information about what FeetWetCoding is and why it was made, you may read the Introduction.


NOTE: This documentation is still being developed--it is incomplete and in many places, inaccurate!


The FeetWetCoding C++ learning framework was put together to be used in a certain way, and if you are a beginning programmer, it is important that you understand how to use it correctly.  Save yourself time and aggravation by reading the following two pages of instructions:


1. Download and install the QtCreator SDK.

2. How to use the FeetWetCoding learning framework.


Once you have read those, you are ready to get started with:


3. The Exercises.


Note:  Every FeetWetCoding exercise source code file will have a page of documentation supporting it.  Not every page of documentation will have source code associated with it.  There is supporting documentation here for subjects related to C++, but which are not actually about C++ programming.
