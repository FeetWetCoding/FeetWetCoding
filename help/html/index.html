FeetWetCoding


FeetWetCoding is a free, open-source software project that tries to make learning C++ fun using Qt.  For more information about what this project is and why it was made, you may read About FeetWetCoding.  Here's what you need to use our software: Requirements.  This project is "alpha" software--it is currently in development and is unfinished!


Easy As 1-2-3!


    1. Get the software.
    2. Learn how to use it.
    3. Do the Exercises.
