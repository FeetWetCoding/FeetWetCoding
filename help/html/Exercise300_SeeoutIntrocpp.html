Chapter 01, Section 02, Exercise 300_SeeoutIntro.cpp


Add seeout lines to make your seeout output match the solution output.


In this exercise, you will learn:


● How to display text in the seeout output area using seeout.


To do this exercise you need to know:


● How to load the fwc project in Qt Creator, and run it.


Your exercise code prints the letters 1 through 5, each on a separate line.  The solution code prints the lines up to 10.  Add seeout lines to your code to print the numbers up through 10.


seeout is useful for printing information as your program runs.  You will see why seeout is a more convenient way to display text in some cases than using fwcDrawText() in Section03, Exercise 040_DrawingVersusSeeout.cpp.


Wait, seeout doesn't look like a function!  Where are the parentheses?


Maybe you noticed that seeout has a different syntax than the functions you are getting used to.  You do not enclose your text string in parentheses with seeout.  This is because seeout is not a function, it is something else called a "stream", which you don't need to worry about right now.  Just notice that this syntax uses the "<<" characters to sort of "send" the text on the right side of the << to seeout.  "<<" is an "operator", similar to how + and - are mathematical operators.  Don't worry if this is confusing, we will do more with operators soon.
