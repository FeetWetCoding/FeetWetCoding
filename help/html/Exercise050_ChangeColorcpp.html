Chapter 01, Section 02, Exercise 050:  ChangeColor


In this exercise, you will learn:


  • How to change the color of text you draw in the drawing area.


To do this exercise you need to know:


  • How to load the fwc project in Qt Creator, and run it.


Once again, in this exercise, we are just changing a parameter in the function fwcDrawText().  In this case, we are changing the fourth parameter, color.  You will change the color BLUE to the color RED.  It is case-sensitive, so Red or red or rED will not work.


BLUE and RED are not the only colors you can use, by the way.  In exercise 520_ColorNames.cpp you will see which color names produce which colors.

When you are done, check the solution code, which lives one directory below your exercise code in the "Solutions" directory, and is named:

  • 050_ChangeColorSOLN.cpp

Always understand the SOLN code before you proceed to the next exercise!


When you are ready, you can proceed to 060_ChangeFontSize.cpp.
