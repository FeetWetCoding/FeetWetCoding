Chapter 01, Section 02, Exercise 250_DrawingImages.cpp


Rearrange the images to look like the solution.


In this exercise, you will learn:


● How to draw images in the drawing area using the fwcDrawImage() function.


To do this exercise you need to know:


● How to load the fwc project in Qt Creator, and run it.

● What functions and parameters are.  ( See Exercise 010_ChangeText.cpp )


Change the x,y parameters for the fwc, spaceship and clock images to match their locations in the solution drawing area.  You will have to add fwcDrawImage() lines to your code for the three extra spaceships in the solution.


The people displayed are (from left to right) Ken Thompson, Dennis Ritchie, Brian Kernighan and Bjarne Stroustrup.  Ritchie and Kernighan created the language C, and Stroustrup created C++.  Thompson is one of the principle creators of Unix, and was also involved in the creation of C.


What other images can you use?


As of 2012Mar24 we are working on our image library.  You can look in the img directory to see what images are available.  Our Attribution page explains where we got these images.  In later exercises we will make use of the images in the img directory.  For now, feel free to play with the images in that directory by adding them to this exercise once you have achieved the solution.  In Section06 fwcTools, you will learn how to scale, rotate and move images (and other graphical objects like circles, lines, etc.) in fwc.


When you are done working on the exercise, or if you get stuck and can't figure it out, check the solution code, which lives one directory below your exercise code in the "Solutions" directory, and is named:

● 250_DrawingImagesSOLN.cpp

Always understand the SOLN code before you proceed to the next exercise!


When you are ready, you can proceed to 300_SeeoutIntro.cpp.
