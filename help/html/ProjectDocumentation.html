A Note About This Documentation


This documentation is currently generated with the helpndoc app.  We considered moving to a wiki-based online form of documentation, but have decided against that at least for now.  Helpndoc allows us to include all the documentation with the source distro, and we feel this is more important to the beginning programmer than any benefits we might get from having the docs online.
