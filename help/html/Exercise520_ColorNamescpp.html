Chapter 01, Section 02, Exercise 520_ColorNames.cpp


Reference Exercise showing available colors.


In this exercise, you will learn:

● Which colors are available to you for use with the fwc drawing tools.


Why you need to know this:

● You will be using these colors in almost all future fwc exercises, so you need to know where to look to find which colors have which names.


To do this exercise you need to know:

● How to load the fwc project in Qt Creator, and run it.


Since this is a "Reference Exercise", there isn't anything for you to do.  Refer back to this code in later exercises when you need to see what the name of a particular color is for use with the fwc drawing tools such as fwcDrawCircle(), fwcDrawText() etc.  So, since you will want to be able to refer back to this code later:


Do not modify the code for this exercise!


When you are ready, you can proceed to 550_FontProportions.cpp.
