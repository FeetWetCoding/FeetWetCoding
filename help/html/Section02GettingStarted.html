Getting Started


Welcome to FeetWetCoding!  This section, "Getting Started" introduces you to the basics of how you will get information from your program onto the screen.  Traditionally, if you started to learn C++, you would get your program's output by using the cout operator.  With FeetWetCoding, you will be able to get output using something very similar that we have setup for you to use, called "seeout."  You will also be able to draw simple graphics on the screen, which you will use in later sections to see what your program is doing in a visual way.


Most C++ texts start off with about 50 pages of discussion of types, variables and basic syntax.  There's a good reason they do that, but with FeetWetCoding, we want you to jump right in and get your feet wet!  So we start off by showing you how to actually make your program do something:  draw stuff on the screen.


Now it turns out that without types, variables and all that other jazz, you can't really do much just by drawing things on the screen with the tools you are going to learn in this section.  But as you learn the other stuff, you will already know how to at least display the new tools and features of C++ on the screen, so that you get some visual sense of what your program is trying to do.
