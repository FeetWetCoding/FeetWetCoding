Chapter 01, Section 02, Exercise 200_DrawingText.cpp


Rearrange the text to look like the solution.


In this exercise, you will learn:

● How to draw text in the drawing area using the fwcDrawText() function.


Why you need to know this:

● Since you will be using the fwcDrawText() function a lot in future fwc exercises, it is good for you to have some more practice with it.


To do this exercise you need to know:

● How to load the fwc project in Qt Creator, and run it.

● What functions and parameters are.  ( See Exercise 010_ChangeText.cpp )


This is the first sort of "pain in the butt" exercise you will have in fwc, although it won't be the last!  :-)  However you may take comfort in the fact that with almost all of the exercises you will do later, you won't be arranging words individually like this!  Just think of it as good practice to help you keep getting used to the x,y drawing space.


The goal here is (as usual) to make your exercise drawing area look like the solution drawing area.  You will need to change coordinates for words so they appear in their correct places on the screen.  You will also have to change the fontsize parameter in some cases.


You'll notice that while you still have a color parameter in this function, you don't have linewidth or solid parameters when you use fwcDrawText().  With text you do have a fontsize parameter which controls how big the font is on the screen.


When you are done working on the exercise, or if you get stuck and can't figure it out, check the solution code, which lives one directory below your exercise code in the "Solutions" directory, and is named:

● 200_DrawingTextSOLN.cpp

Always understand the SOLN code before you proceed to the next exercise!


When you are ready, you can proceed to 250_DrawingImages.cpp.
