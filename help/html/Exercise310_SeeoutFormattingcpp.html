Chapter 01, Section 02, Exercise 310_SeeoutFormatting.cpp


Add \n and spaces to your seeout text so that your output matches the solution.


In this exercise, you will learn:

● How to make newlines and add spaces to seeout output.


Why you need to know this:

● It is important for your text output to be readable, and so you need to know how spaces and newlines work when using seeout.


To do this exercise you need to know:

● How to load the fwc project in Qt Creator, and run it.


Each number should appear on it's own line, and the sentence should have spaces between the words.


Notice that seeout doesn't put things on separate lines just because they are on separate lines in your code.  The "\n" character is what forces seeout to generate a newline.  You can mix the "\n" character in anywhere in your text to generate a newline.


When you are done working on the exercise, or if you get stuck and can't figure it out, check the solution code, which lives one directory below your exercise code in the "Solutions" directory, and is named:

● 310_SeeoutFormattingSOLN.cpp

Always understand the SOLN code before you proceed to the next exercise!


When you are ready, you can proceed to 400_OnYourOwn.cpp.
