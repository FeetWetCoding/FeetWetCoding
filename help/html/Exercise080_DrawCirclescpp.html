Chapter 01, Section 02, Exercise 080_DrawCircles.cpp


In this exercise, you will learn:


● How to draw circles on the drawing area using the fwcDrawCircle() function.

● The solid parameter determines whether the circle is filled in with color or not.

● The solid parameter accepts two values:  true and false.

● The solid parameter is optional.

● When objects are drawn in the same place on the screen, the last object to get drawn is "on top" of the previous objects.


To do this exercise you need to know:


● How to load the fwc project in Qt Creator, and run it.

● What functions and parameters are.  ( See Exercise 010_ChangeText.cpp )


This exercise draws three circles.  The RED circle is superimposed "on top" of the BLUE and BLACK circles, because it is drawn last.


In the SOLN you can see that the RED circle is in the same location as it is in your exercise, but the BLUE and BLACK circles are in different locations.  You will need to change the x,y coordinates for the BLUE and BLACK circles to relocate them to the positions the SOLN code displays them at.  All of the circles have the same radius in the SOLN code as they do in your exercise code.


You will also see that the BLUE and BLACK circles are superimposed on the RED circle in the SOLN.  This is a puzzle: for you to figure out how to make that happen in your exercise code.


   // USAGE: fwcDrawCircle(x, y, radius, color, linewidth, solid);
   // (solid is an optional parameter, you can use it or leave it out.)
   fwcDrawCircle(200, 300, 80, BLUE, 1);


The USAGE comment explains what parameters you pass to fwcDrawCircle()


The next line draws a circle with it's center at 200,300 with a radius of 80px in color BLUE, having a linewidth of 1px.


When you are done working on the exercise, or if you get stuck and can't figure it out, check the solution code, which lives one directory below your exercise code in the "Solutions" directory, and is named:

● 080_DrawCirclesSOLN.cpp

Always understand the SOLN code before you proceed to the next exercise!


After you have achieved the solution, practice using fwcDrawCircle() to draw different circles.  Vary the position, radius, color, linewidth and fill (solid), until you feel comfortable using the fwcDrawCircle() function.  This is your code and your learning, so always feel free to have fun and experiment!  The more you practice, the more your learning will sink in.


When you are ready, you can proceed to 090_DrawRectangles.cpp.
